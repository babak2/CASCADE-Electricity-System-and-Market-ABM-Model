createHouseholdProsumer

public HouseholdProsumer createHouseholdProsumer(double[] miscDemandProfileArray, boolean addNoise, int occupancyNb)
Creates a household prosumer with a basic consumption profile as supplied with or without added noise NOTE: The addNoise option is unsafe as implemented - under a certain combination of random factors, it can give negative demands!! DO NOT USE
Parameters:
baseProfile - - an array of the basic consumption profile for this prosumer (kWh per tick)
addNoise - - boolean specifying whether or not to add noise to the profile
occupancyNb - - number of occupancy per household, if Consts.RANDOM is passed, it will done randomly
